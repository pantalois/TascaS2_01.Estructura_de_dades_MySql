# TascaS2_01.Estructura_de_dades_MySql

## Description

- Practice and demonstration of basic and advanced data structures.  
- Integration with MySQL for persistent data storage and retrieval.  
- Designed for learning and applying database concepts in combination with data structures.  

## Level_1/exercici_1  
This project consists of creating a database for an optician called **"Cul d'Ampolla"**.  
The database stores information about suppliers, glasses brands, customers, and sales staff.  
It also includes specific details such as referrals, lens specifications, and employee participation in sales.  

## Level_1/exercici_2  
This project involves designing a database for a pizzeria with a delivery service.  
The database contains information about customers, addresses, orders, and product details such as pizzas, hamburgers, and drinks.  
It also manages stores, employees, and delivery assignments.  

## Technologies Used

- **MySQL** – Relational Database Management System  
- **SQL** – Structured Query Language  
- Tools for database design such as **MySQL Workbench**
